## 已上线

### 🎨 微信表情包工具箱

面向表情包创作者的全套在线改图工具：

| 功能 | 说明 |
|------|------|
| ✂ 裁剪缩放 | 240×240 表情主图 / 120×120 缩略图 / 50×50 图标 / 自定义尺寸 |
| 🖊 文字引擎 | 白字黑描边 / 黄字红描边 / 竖排咆哮 / 彩色阴影 / 小字备注 |
| 💬 对话气泡 | 微信绿气泡 / 微信白气泡 / 漫画气泡 / 爆炸气泡 / 想法气泡 |
| 🖼 表情拼贴 | 左右对比 / 上下对比 / 四宫格 / 对角拼贴 |
| 🎭 搞怪滤镜 | 黑白线稿 / 高饱和老年 / 像素风 / 毛边粗糙 / 浮雕 |
| 🔲 白底处理 | 杂底→纯白 / 透明→白底（防微信渲染黑底） |
| 🔄 格式转换 | PNG / JPEG / WebP |
